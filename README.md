# Set of cucumber scenarios for various Hashgraph APIs

## Prerequisites

To run the tests you need to have the following env variables set:

```env
ACCOUNT_ID=
PRIVATE_KEY=
```

## Running the tests

To run the tests

```bash
./gradlew clean test
```
